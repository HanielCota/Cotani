# Security Policy

## Supported Versions

Only the latest stable release of Cotani is supported for security updates.

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |
| < 1.0.0 | :x:                |

## Reporting a Vulnerability

If you discover a security vulnerability within Cotani, please do **NOT** open a public GitHub issue.

Instead, please report the vulnerability directly to the project owner:

- **Contact**: Haniel Cota
- **GitHub**: [@HanielCota](https://github.com/HanielCota)

Please include:
1. Description of the vulnerability and affected module(s).
2. Steps to reproduce or proof-of-concept.
3. Potential impact on Paper servers or applications.

We will review the issue promptly and publish a patch in a timely manner.
